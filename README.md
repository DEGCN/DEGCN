# DEGCN
Code for paper Dual Enhanced Graph Convolutional Networks for Aspect-based Financial Sentiment Analysis

## Requirements
* Python 3.9
* PyTorch 1.12
* SpaCy 3.3.1
* numpy 1.21.5
* entmax 1.0
* argparse 1.4.0
* scikit-learn 1.0.2

## Usage

* Training with the script file [run.sh](./run.sh)
```bash
bash run.sh
```
